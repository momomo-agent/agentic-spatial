Capabilities

What it sees.

More than geometry. It understands what's happening.

🧠

Emotion Detection

Per-person emotional state from facial expression and body language. Tracks shifts over time — focused → contemplative → emotional.

🔄

Continuous Mode

Feed frames over time. Detects who appeared, disappeared, moved, changed pose, or shifted emotion. Stable ID tracking across frames.

👥

Social Graph

Who's interacting with whom. Face-to-face conversations, group dynamics, isolated individuals. The invisible social map.

⚓

Anchor System

Large objects anchor the spatial skeleton. Everything else is positioned relative to them. Consistent across runs.

🎯

Ensemble Mode

Run 3× parallel analyses, merge by coordinate matching. Median coordinates, majority vote. 89% cross-run consistency.

📹

Camera Awareness

Who's looking at which camera. Per-camera visibility analysis. Blind spot detection. Full coverage mapping.

Usage

Two lines to start.

// Single analysis import { reconstructSpace } from 'agentic-spatial' const scene = await reconstructSpace({ images, apiKey, model: 'claude-sonnet-4', ensemble: 3 // optional: 3× parallel for higher consistency }) scene.people // position, emotion, activity, gaze scene.behaviors // meeting, presenting, solo_work scene.coverage // camera visibility, blind spots

// Continuous mode — feed frames over time import { SpatialSession } from 'agentic-spatial' const session = new SpatialSession({ apiKey }) await session.analyze(initialImages) // first frame await session.update(newImages) // detect changes session.state.changes // appeared, disappeared, moved, emotion_changed session.getHistory() // full timeline
